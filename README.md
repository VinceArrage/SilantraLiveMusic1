Overview

This project is designed to capture the sophisticated "Modern Noir" aesthetic of Silantra’s brand. It serves as a digital portfolio and booking engine for their 2026-2027 season, highlighting their versatility in Jazz and Pop arrangements.
Key Features

    -Formation Showcase: Dedicated sections for Duo, Trio, Quartet, and Quintet ensembles.

    -Service Highlighting: Tailored content for Weddings, Corporate Events, Cocktails, and Galas.

    -Booking Engine: A streamlined inquiry form for lead generation.

    -Responsive Design: Optimized for high-end mobile and desktop experiences.

    -Regional SEO: Targeted for the MTL-OTT-GAT tri-city area.
